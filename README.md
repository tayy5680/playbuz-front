# PlayBUZ Front

## node version support

```
v16.4.0
```

## 專案套件初始化

```bash
npm run setup
```

### 本地開發環境

```bash
npm run serve
```

### 打包專案

```bash
npm run build
```

### 生成 sitemap.xml

```bash
npm run sitemap
```

### 清除本地 tag

```bash
npm run tag
```

### 更新翻譯 json

```bash
npm run xlsx
```
